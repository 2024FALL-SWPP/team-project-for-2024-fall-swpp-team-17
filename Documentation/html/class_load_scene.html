Detailed Description

Manages scene loading and transitions within the game. Provides methods for loading specific scenes, the next scene, or reloading the current scene.

This class uses Unity's SceneManager to handle scene transitions. Ensure all scenes are added to the Build Settings to enable proper transitions.
