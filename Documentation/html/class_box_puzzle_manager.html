Detailed Description

Manages the logic for a box puzzle, including the activation of boxes, locks, and plates. Handles puzzle progression, resetting, and clearing.

This class implements PuzzleInterface and provides functionality for:

  • Tracking the positions of keyboxes and resetting them.
  • Activating locks and plates based on player interaction.
  • Clearing the puzzle and triggering corresponding visual effects.
