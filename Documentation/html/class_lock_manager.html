Detailed Description

Manages the behavior of a lock within a puzzle. Handles unlocking logic when the correct key collides with the lock and sends a signal to the puzzle interface upon unlocking.

Each lock has a unique ID assigned via SetLockID and interacts with the PuzzleInterface to notify the puzzle of unlock events.
