Detailed Description

Manages the fade-in effect for scenes using a UI Image overlay. Gradually decreases the alpha of the fade image to create a smooth transition effect.

Attach this script to a GameObject with an Image component assigned to fadeImage. Ensure the fade image covers the entire screen for a seamless fade effect.
