Detailed Description

Handles sprite changes for a UI button based on user interactions. Implements hover, click, and release states using defined sprites.

Requires an Image component and three sprites:

  • defaultSprite: Default button state.
  • activeSprite: Hover/active button state.
  • pushedSprite: Clicked button state.
