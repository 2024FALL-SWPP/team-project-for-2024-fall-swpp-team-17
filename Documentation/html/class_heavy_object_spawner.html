Detailed Description

Spawns a heavy object above the player at specified intervals when the player enters a defined zone. Automatically destroys the previously spawned object before creating a new one.

This script requires:

  • A boxPrefab to spawn.
  • A player transform for determining the spawn location.
  • A spawnPosition to define the z-coordinate threshold for triggering the spawn logic.
